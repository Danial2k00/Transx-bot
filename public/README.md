# Public Assets Folder

## Robot Holographic Display Image

Place your robot image here with the filename:

- `robot-holographic-display.png` (preferred)
- `robot-holographic-display.jpg`
- `robot-holographic-display.webp`

### Image Specifications

**Recommended Dimensions:**

- Width: 800px - 1200px
- Height: 600px - 900px
- Aspect Ratio: 4:3 or 16:9

**File Size:**

- PNG: Under 500KB (optimized)
- JPG: Under 300KB
- WebP: Under 200KB (best performance)

**Image Requirements:**

- Transparent background (PNG/WebP) preferred for better integration
- High resolution for crisp display on retina screens
- Holographic/futuristic aesthetic to match the AI trading bot theme

### Usage

The image will automatically appear in the hero section of the homepage. It will be displayed alongside the animated bot visualization and trading interface elements.

### Other Assets

This folder can also contain:

- Logo files (`next-gen-bot-logo.png`)
- Additional hero section graphics
- Static images used throughout the website

All files in this folder are publicly accessible via the `/` path in the application.
